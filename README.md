# KMKB Application

Kendali Mutu Kendali Biaya (KMKB) Application based on clinical pathways.

## Project Structure

This repository contains the implementation of the KMKB application as specified in the BRD and Implementation Plan documents.

## Documentation

All project documentation can be found in the `documentation/` folder:

- `BRD_KMKB.md` - Business Requirements Document
- `IMPLEMENTATION_PLAN.md` - Implementation plan and timeline
- `1.1_setup_environment.md` - Environment setup guide
- `1.1.1_setup_laravel_framework.md` - Laravel framework setup guide

## Directories

- `src/` - Main source code for the application
- `documentation/` - All project documentation

## Getting Started

Follow the setup guides in the documentation folder to get the development environment running.
